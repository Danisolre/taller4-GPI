# Taller 4 — Workflow reproducible para análisis de contratos públicos

Este repositorio contiene la implementación del flujo de trabajo desarrollado en el Taller 4 de Gestión de Proyectos y Flujos de Trabajo Colaborativo. El objetivo es construir un pipeline reproducible de análisis de datos, aplicar control de versiones con GitHub y documentar el proceso mediante diagramas de workflow.
